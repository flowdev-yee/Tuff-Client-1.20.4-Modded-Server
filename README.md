Eaglercraft (Tuff Client) 1.20.4 Modded Server

You can join this server on Tuff Client and experience all the modern features!

You can also put mods in the mods folder in the server directory. Note that not all mods will work!

How to host:
Make codespace
Make a new terminal
In terminal one, type 'cd server && java -jar server.jar'
Wait until there is a 1 next to "Ports"
In terminal two, type 'cd velocity && java -jar velocity.jar'
Wait until there is a 3 next to "Ports"
Set Port 8081 visibility to "Public"
Press on the globe next to 8081 when hovering over it
Copy the wss:// address
Join it in Tuff Client
Enjoy!

You can play Tuff Client at: https://test.tuffis.online
