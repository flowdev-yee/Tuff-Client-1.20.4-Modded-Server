Eaglercraft (Tuff Client) 1.20.4 Modded Server

You can join this server on Tuff Client and experience all the modern features!

You can also put mods in the mods folder in the server directory. Note that not all mods will work!

How to host:

1. Make codespace
2. Make a new terminal
3. In terminal one, type 'cd server && java -jar server.jar'
4. Wait until there is a 1 next to "Ports"
5. In terminal two, type 'cd velocity && java -jar velocity.jar'
6. Wait until there is a 3 next to "Ports"
7. Set Port 8081 visibility to "Public"
8. Press on the globe next to 8081 when hovering over it and copy the wss:// address in the site
9. Join it in Tuff Client
10. Enjoy!

You can play Tuff Client at: https://test.tuffis.online
